HTML
----
1.HTML stands for Hyper Text Markup Language.
2.Tag or Element
----------------
Text placed in between left angle brace '<' and right angle brace '>' is called a tag or element.
syntax(set of rules)
--------------------
open(start) tag				close(end) tag
----------------------------------------------------------------
<html>---------------------------------------</html>
<p>------------------------------------------</p>

4.HyperText = HyperLink = Navigation(Moving from one page to another page)

5.Markup Language : Syntax which consists Tags or Elements / Attributes and Parameters

6.HTML was started developing in 1990's by Tim Berner's Lee
HTML4.01 was released in the year 1999

7.W3C(World Wide Web Consortium - w3.org) was founded in 1994 by Tim Berner's Lee

8.W3C provides guidelines / standards for those who are writing html.

9.HTML is a case insensitive language.
ex : <HTML> = <htML> = <html> (lowercase)
ex : <BODY BGCOLOR=RED> = <body bgcolor="red"> or <body bgcolor='red'>

10.HTML is a global language i.e, it can be understood by all the available browsers.

11.We save it with .html or .htm extension.

12.Latest version of HTML is HTML5

13.HTML5=HTML4 + XHTML + ADVANCED JAVASCRIPT

15.Types of Tags
----------------
1.Container or Paired tag
-------------------------
A tag which consists both opening and closing
ex : <p>---------------</p>
     <html>------------</html>
     <b>----------------</b>
/ - forward slash

2.Empty or Unpaired tag
-----------------------
A tag which consists only opening
ex : <br>, <hr>, <img>......
NOTE
----
HTML, HTML5, ANGULAR : <br>, <hr>, <img>......
XHTML, REACTJS : <br/>, <hr/>, <img/> (self closed)

XHTML
-----
1.XHTML is a stricter and cleaner version of html : definition
2.XHTML stands for eXtensible Hyper Text Markup Language
3.XHTML is recommended by W3C(World Wide Web Consortium)
Differences between HTML and XHTML
----------------------------------
HTML					XHTML
-----------------------------------------------------------------
<HTML>					<html>
<P>-----</p>				<p>--------</p>
<p>-----				<p>--------</p>
<b><i>------</b></i>			<b><i>-----</i></b>
<TD WIDTH=100>				<td width="100">
					<td width='100'>
<hr>					<hr/>
<img>					<img/>
<ul>					<ul>
  <li>-----</li>                           <li>-----</li>
  <li>---                                  <li>---
     <ul>                                     <ul>
        <li>----</li>                            <li>----</li>
     </ul>                                    </ul>
  <li>-----</li>                            </li>
</ul>					    <li>-----</li>
                                         </ul>
NOTE
-----
<hr>, <br>, <img> : html, html5, angular
<hr/>, <br/>, <img/> - self closed : XHTML, ReactJS
